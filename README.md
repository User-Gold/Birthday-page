# 💖 Happy Birthday, Madhu! 💖

A personalized, interactive, and beautifully designed birthday celebration web page. It is built as a special digital gift featuring magical interactions and heartfelt surprises.

## ✨ Features

- **💌 The Secret Letter:** A hidden modal that types out a personalized birthday message dynamically, complete with a revealing sticker and floating hearts.
- **🎂 The 3D Cake:** An animated CSS 3D cake with a flickering candle you can actually "blow out" by clicking the flame—followed by a cinematic dark-to-party-mode transition.
- **🎁 The Scratch Card:** A realistic digital gold-foil scratch card built with the HTML5 Canvas API. Scratching exactly 20% magically clears the rest, popping up a final secret message and showering the screen in confetti.
- **🎈 Micro-Animations:** Floating balloons, beating hover effects, rotating text, and dynamic bubbles perfectly round out the polished user experience.
- **📱 Fully Responsive:** Carefully modeled for an elegant look on both mobile phones and desktop computers.

## 🛠️ Technology Stack

- **HTML5:** Clean and semantic structure.
- **Vanilla CSS3:** Advanced 3D Transforms, Keyframe Animations, Flexbox, and complex Gradients.
- **JavaScript & jQuery:** DOM manipulation, interactive event listeners (for the scratch canvas and modals), and timing logic.
- **Canvas Confetti:** For explosive, celebratory particle effects.

## 🚀 How to Run Locally

You can run this project effortlessly using any local development server. 

If you have Node.js installed, simply run the following from your terminal in the project directory:

```bash
npx serve .
```

Then, open your browser and visit `http://localhost:3000`.
